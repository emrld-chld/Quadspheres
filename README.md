Quadspheres simply adds a "Quads" submenu into Shift + A menu.

Menu items:

Quadsphere (Remembers your last used settings)
Quadsphere + Subdivision Surface 3
Quadsphere + Subdivision Surface 4
Quadsphere + Subdivision Surface 3 (Applied)
Quadsphere + Subdivision Surface 4 (Applied)
Quadsphere + Multiresolution 3
Quadsphere + Multiresolution 4


Installation:
- go to Blender > Preferences > Addons > Install from disk
- select the quadsphere_addon.py
